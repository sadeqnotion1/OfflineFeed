# OfflineFeed — Design System

> Generated by applying the **UI UX Pro Max** reasoning flow
> (product type → pattern → style → colors → typography → effects → anti-patterns → checklist)
> to OfflineFeed, a Python-backed **Telegram Desktop clone** built in **Qt (PySide6 + QML)**.

## 1. Product match
- **Product type:** Desktop messaging / chat client (Telegram Desktop clone)
- **Pattern:** Multi-pane desktop messaging shell
  - Folder rail (68px) → Chat list (280–420px) → Chat view (fluid) → Info panel (slide-in 340px)
- **Primary surfaces:** title bar, folder rail, chat list, chat view (header + pinned bar + bubbles + composer), info panel

## 2. Style
- **Selected style:** Dark Mode (OLED) + Micro-interactions, matching Telegram's native design language
- **Why:** night-friendly comms apps benefit from low-luminance surfaces; Telegram parity demands its exact tokens
- **Light mode** ships as a first-class cross-fading alternative

Avatars use Telegram's 7 per-user gradient palettes (see `Avatar.qml`).

## 4. Typography
- **UI font:** Roboto (Telegram Desktop's UI typeface)
- **RTL / Persian fallback:** Vazirmatn
- Sizes: 11 (meta) · 12–13 (secondary) · 14–15 (body/name) · 18 (panel title)

## 5. Key effects
- Theme cross-fade: `ColorAnimation 180ms`
- Row hover/selection: `ColorAnimation 120ms`
- Info-panel slide: `NumberAnimation 200ms Easing.OutCubic`
- Bubble appear: `NumberAnimation 160ms`
- Send ↔ mic morph on composer
- All easing `OutCubic`, durations 120–200ms to match Telegram's feel

## 6. Anti-patterns avoided
- ❌ Emoji used as icons → ✅ all icons are theme-tinted **SVGs** (`Icon.qml` + `assets/icons/`)
- ❌ Flat single-color avatars → ✅ per-user gradient avatars
- ❌ Harsh/instant state changes → ✅ 120–200ms eased transitions
- ❌ Neon/AI purple-pink gradients (off-brand for a comms tool)

## 7. Pre-delivery checklist
- [x] No emojis as icons (SVG line icons, Feather/Lucide-compatible)
- [x] Hover states with smooth transitions (120–200ms)
- [x] Dark + light themes, both with ≥4.5:1 text contrast
- [x] Keyboard-focusable text fields
- [x] RTL/Persian support hook (LayoutMirroring + Vazirmatn)
- [x] Resizable, frameless native window with min sizes
